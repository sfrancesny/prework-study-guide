Git

  • git is a version control system that lets you manage and keep track of each version of your project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "[your message] commits your staged content as a new commit snapshot
  • git config --global user.name [firstname lastname] sets a name that is identifiable for credit when review version history
  • git config --global user.email [valid email] sets an email address that will be associated with each history maker
  • git log shows all commits in the current branch's history
  • git merge [alias][branch] merges a remote branch into your current branch to bring it up to date
  • git push [alias][branch] transmit a local branch commit to the remote repository branch
  • git pull this allows you to fetch and merge any commits from the tracking remote branch
