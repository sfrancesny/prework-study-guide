CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • declarations contain two important components: the CSS property we want to apply and the value of the property
  • the CSS box model applies to block boxes and defines how different parts of a box work together to create box that you can see on a page
  • content box is the area where your content is displayed
  • border box wraps the content and any padding
  • disply: inline-block is a special value of display which provides a middle ground between inline and block

Git

  • git is a free and open source version control system that's responsible for everything Github related that happens locally on your computer
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git config --global user.name “[firstname lastname]” sets a name that is identifiable for credit when review version history
  • git config --global user.name "[firstname lastname]" sets an email address that will be associated with each history marker
  • git add [file] allows you to add files while retaining the changes in working directory
  • git commit -m "[descriptive message]" commits your staged content as a new commit snapshot
  • git merge [brach] merges the specified branch's history into the current one
  • git push [alias][branch] transmit local branch commits to the remote repository branch
  • git pull fetches and merges any commits from the tracking remote branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a powerful programming language that can add interactivity to a website. It was invented by Brendan Eich.
  • Browser Application Programming Interfaces (APIs) built into web browsers, providing functionality such as dynamically creating HTML and setting CSS styles; collecting and manipulating a video stream from a user's webcam, or generating 3D graphics and audio samples is an example of a tool created from JavaScript. Variables are containers that store values. You start by declaring a variable with the let keyword, followed by the name you give to the variable.
  • One important thing to remember JavaScript is case sensitive.
  • A string is a sequence of text. To signify that the value is a string, enclose it in single or double quote marks.
  • A number is exactly what it says it is, a number. Numbers don’t have quotes around them.
  • Boolean is a true/false value. They do not require quotation marks like string variables.
  • An array is a structure that allows you to store multiple values in a single reference.
  • Comments are snippets of text that can be added along with code.
  • An operator is a mathematical symbol that produces a result based on two values (or variables).
  • Conditionals are code structures used to test if an expression returns true or not. A common form of conditionals is the if...else statement.
  • A function is a set of statements that performs a task or calculates a value.
